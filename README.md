# Live Link

# Technologies

- Database management
	- MongoDB
  - Mongoose
- Front-end development
	- React
  - Redux
  - React Testing Library
  - JS-Cookie
  - CSS Grid
	- CSS
- Back-end development
	- Express.js
  - Babel
  - Nodemon
	- JWT
	- Dotenv
  - CORS
