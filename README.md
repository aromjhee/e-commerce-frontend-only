- Database management

  - Mongoose
- Front-end development
	- React

  - React Testing Library
  - JS-Cookie
  - CSS Grid

- Back-end development
	- Express.js
  - Babel
  - Nodemon
	- JWT
	- Dotenv
  - CORS
