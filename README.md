# Introduction
imma-Zone is an e-commerce clone.

# Live Link

[imma-Zone](https://e-commerce-5555-frontend.herokuapp.com/)

# Technologies

- Database management
	- MongoDB
  - Mongoose
- Front-end development
	- React
  - Redux
  - React Testing Library
  - JS-Cookie
  - CSS Grid
	- CSS
- Back-end development
	- Express.js
  - Babel
  - Nodemon
	- JWT
	- Dotenv
  - CORS
